## Requirements

> **Note:** Please follow bellowing requirements of environments

| Dependency    | Version  |
| --------------|----------|
| Node          | 14.17.4  |
| Xcode         | 13.4.1   |
| Cocoapods     | 1.10.0   |
| Ruby          | 2.6.8    |

## How To Start
- `npm install`
- `cd ios && pod install`
- `cd .. && npm run start`
- `npm run ios`
